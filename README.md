# AI Summary CLI Tool

## Overview
A simple command-line tool that converts unstructured text into a structured summary using an LLM API.

## Features
- Accepts input via:
  - Direct CLI text
  - File path
  - stdin (piped input)
- Outputs:
  - One sentence summary
  - Three key points
  - Sentiment analysis

```bash
git clone https://github.com/your-username/ai-summary-cli.git
cd ai-summary-cli
